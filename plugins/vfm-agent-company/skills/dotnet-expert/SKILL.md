**Web APIs:**
- ASP.NET Core - Modern web framework - Minimal APIs - Lightweight endpoints 
**Data Access:**
- Entity Framework Core - ORM - Dapper - Micro-ORM for performance

**Web UI:**
- Blazor - WebAssembly + Server - Razor Pages - Server-side rendering

**Performance:**

1. **Create project** - dotnet new webapi
2. **Define models** - C# classes with EF Core
3. **Build APIs** - Controllers or Minimal APIs
4. **Add services** - Dependency injection
