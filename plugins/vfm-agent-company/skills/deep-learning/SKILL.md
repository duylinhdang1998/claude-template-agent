## Overview

Deep Learning at Google scale - neural networks powering billions of users across Search, YouTube, Photos, Assistant. Not toy models - production ML at the highest scale.

## When to Use What

**Computer Vision:**
- Image classification → CNNs - Object detection → Faster R-CNN, YOLO
- Vision Transformers → ViT

**Natural Language:**
- Text understanding → Transformers - BERT, T5, PaLM models
- Fine-tuning for tasks

**Sequence Modeling:**
- Time series → RNNs, LSTM - Audio/speech → Wav2Vec, Conformer

## Core Workflow

1. **Data preparation** - Clean, augment, split datasets
2. **Model selection** - Choose architecture for task
3. **Training** - Use TPUs, distributed training 4. **Evaluation** - Rigorous metrics, validation
5. **Optimization** - Hyperparameter tuning, architecture search
6. **Deployment** - TensorFlow Serving, optimization 7. **Monitoring** - Track performance, detect drift

## Best Practices

1. Start with pre-trained models - Transfer learning
2. Use TPUs for large models - Cost-effective training
3. Monitor training closely - TensorBoard, W&B
4. Validate rigorously - Multiple metrics, holdout sets
5. A/B test in production - Measure real impact
